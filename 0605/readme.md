## Numpy

대규모 배열과 행렬 연산에 필요한 다양한 함수 제공

- NumPy 배열 생성 및 조작

      np.array(): 리스트나 튜플로부터 배열 생성
  
      np.arange(), np.linspace(): 연속된 값으로 배열 생성

            => arange 함수 : 파이썬의 range함수의 배열 버전
  
      배열의 형태 변경: reshape(), flatten() 등


* 속성 탐색

  array.shape : 배열의 크기

  array.size : 배열의 총 요소 수

  array.dtype : 배열 요소의 데이터 유형

  array.ndim : 몇 차원인지
  
  array.itemsize : 메모리에서 차지하는 바이트 수
  
  - 연산

    array (-,+,/,*) array 2 연산 가능


  - 슬라이싱, 인덱싱
 
    2차원 배열이라면 array[1,2]로 1행의 2열에 있는 수를 출력할 수 있다

    행 하나만 출력시 [2] 가능, 열 하나만 출력시 [:,2] => :를 사용해야 오류가 발생하지 않음

- 파이썬 리스트와 넘파이 배열 차이점

  * 데이터 타입

    리스트 : 다양한 타입 가능

    넘파이 배열 : 모든 요소가 동일해야함

  * 메모리 사용

    리스트: 비연속적으로 메모리에 요소 저장, 각 요소는 객체로 별도의 메모리 공간 차지

    넘파이 배열 : 모든 요소가 연속된 메모리 블록에 저장, 이로인해 데이터 접근과 조작을 더 빠르고 효율적으로 하게 함

  * 성능

    리스트 : 연산 속도가 느릴 수 있다 (다양한 타입의 데이터 처리가능)

    넘파이 배열 : 수학적이고 과학적인 계산에 최적화 되어있어 대량의 데이터를 다룰 때 높은 성능
    
  * 크기 조정

    리스트 : 요소를 추가하거나 제거하면 크기가 자동으로 변경

    넘파이 배열 : 고정된 크기 => 크기 변경하려면 새 배열 생성 후 기존 데이터 복사
    
  * 다양한 기능과 연산

    리스트 : 기본적인 연산과 기능을 제공

    넘파이 배열 : 다차원 배열 지원, 광범위한 함수와 메서드 제공
    
  * 용도
 
    리스트 : 다양한 타입의 데이터를 유연하게 관리할 때 사용

    넘파이 배열 : 과학 계산, 데이터 분석, 대규모 수치 연산 등 사용

  - 3차원

            첫 번째 차원 (Depth): 최상위 레벨에 3개의 배열
    
           두 번째 차원 (Rows): 각 최상위 배열은 2개의 하위 배열
    
           세 번째 차원 (Columns): 각 하위 배열은 5개의 요소
 
  - 배열 생성 및 초기화

      원하는 shape로 배열을 설정하고 각 요소를 특정 값으로 초기화하는 zeros, ones, full, eye 함수 

      파라미터로 입력한 배열과 같은 shape의 배열을 만드는 zeros_like, ones_like, full_like 함수
    
    * zeros, zeros_like
 
         입력을 튜플로 받음 => 2차원 이상으로 만들시 괄호 두 번 사용
 
            np.zeros((3,6))
 
            np.zeros(10)
 
            np.zeros_like(배열)
      
    * ones, ones_like
 
            np.ones((2,3,4)))
 
            np.ones_like(배열)
      
    * full_like
   
            np.fill_like(배열,원하는 수)

   정수의 범위와 메모리 사용량

- zeros 함수 vs empty 함수

  zeros : shape과 dtype을 입력받아 0으로 초기화된 배열을 만들어준다

  empty : shape과 dtype을 입력 받아 초기화되지 않은 배열을 만들어준다

  => 특정 크기와 데이터 타입을 가진 새로운 배열을 빠르게 생성. 배열의 요소는 임의의 값으로 채워진다

  => 메모리에 이미 존재하는 값으로 채워지므로 구조를 빨리 만들기 위해서 사용한다
